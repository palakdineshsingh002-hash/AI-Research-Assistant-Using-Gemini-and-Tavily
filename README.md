_# AI Research Assistant Using Gemini and Tavily

## Overview

AI Research Assistant is a Python-based application that combines the power of Google's Gemini AI and Tavily Search API to perform intelligent web research and generate concise summaries

The assistant searches the web for relevant information and uses Gemini to analyze and present the results in a user-friendly format.

---

## Features

- Real-time web search using Tavily API
- AI-powered summarization using Gemini
- Fast and accurate research results
- Simple and lightweight Python implementation
- Easy to customize and extend

---

## Technologies Used

- Python
- Google Gemini API
- Tavily Search API
- Python Dotenv

## Installation

### Clone the Repository

```bash
git clone https://github.com/your-username/AI-Research-Assistant-Using-Gemini-and-Tavily.git
```

### Move to the Project Directory

```bash
cd AI-Research-Assistant-Using-Gemini-and-Tavily
```

### Install Dependencies

```bash
pip install google-generativeai tavily-python python-dotenv
```

---

## API Configuration

Create a `.env` file and add your API keys:

```env
GEMINI_API_KEY=your_gemini_api_key
TAVILY_API_KEY=your_tavily_api_key
```

---

## Run the Project

```bash
python main.py
```

---

## How It Works

1. User enters a research query.
2. Tavily searches the internet for relevant information.
3. Search results are collected and processed.
4. Gemini analyzes the information.
5. A concise research summary is generated and displayed.
